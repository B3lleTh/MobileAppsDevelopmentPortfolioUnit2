# 📱 Mobile Apps Development Portfolio - Unit 2

Este repositorio forma parte del **Portafolio de Desarrollo de Aplicaciones Móviles (Unidad 2)**.  
Su objetivo es recopilar ejercicios prácticos en **Dart** que refuercen conceptos fundamentales de programación, estructuras de datos y manejo de excepciones, aplicados al desarrollo móvil con **Flutter**.

---

## 📂 Contenido del repositorio

El repositorio incluye los siguientes archivos y carpetas:

- **20c_factorial.dart**  
  Algoritmo para calcular el factorial de un número entero. Ejemplo clásico de recursividad y bucles.

- **21_suma.dart**  
  Programa que realiza operaciones de suma, útil para comprender entrada/salida y operaciones básicas.

- **22_stack.dart**  
  Implementación de una **pila (stack)** en Dart. Incluye métodos `push`, `pop` y recorrido de elementos.

- **23_queue.dart**  
  Ejemplo de una **cola (queue)**, mostrando cómo manejar estructuras FIFO en Dart.

- **24_try_catch.dart**  
  Demostración del manejo de excepciones en Dart utilizando `try`, `catch` y `finally`.

- **equipo/**  
  Carpeta destinada a documentación adicional o información de los trabajos en equipo.

---

## 🎯 Objetivos de aprendizaje

Este portafolio busca:

- Practicar la sintaxis y características principales de **Dart**.  
- Comprender y aplicar **estructuras de datos fundamentales** (pila y cola).  
- Implementar algoritmos básicos como factorial y suma.  
- Desarrollar habilidades de **manejo de errores y excepciones**.  
- Preparar bases sólidas para el desarrollo de aplicaciones móviles con **Flutter**.

---

## 🛠️ Requisitos previos

Para ejecutar los programas necesitas:

- **Dart SDK** instalado en tu computadora.  
- Editor recomendado: **Visual Studio Code** o **Android Studio**.  
- Conocimientos básicos de programación estructurada y lógica de algoritmos.

---

## ▶️ Ejecución de los programas

Cada archivo puede ejecutarse de manera independiente desde la terminal:

```bash
dart nombre_del_archivo.dart
```
Ejemplo : ``` dart 20c_factorial.dart ```
